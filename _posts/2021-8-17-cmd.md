## 1、停止占用端口的应用

### 查找到占用端口的进程号PID

```shell
netstat -ano | findstr 80
```

假设输出：

```
TCP    0.0.0.0:8501           0.0.0.0:0              LISTENING       2692
```

### 停止进程

```shell
taskkill /F /PID 2692
```
